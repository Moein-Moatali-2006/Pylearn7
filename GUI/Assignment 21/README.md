# Song:
Library PySynth

This is a library for making music.
## how install ?
1-Clone the repository:
```
git clone https://github.com/mdoege/PySynth.git
```
2-Enter the directory (cd PySynth) and run
```
cd PySynth
```
3-next:
```
python3 setup.py install
```
### how run ?
```
main.py
```
